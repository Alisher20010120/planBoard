# 🗂 PlanBoard

**PlanBoard** — bu jamoa bilan tasklarni boshqarish uchun yaratilgan platforma. Har bir taskga foydalanuvchi biriktirish, deadline belgilash, comment yozish imkonini beradi. Trello'ga o'xshash loyiha bo‘lib, backend qismi `Spring Boot` asosida yaratilgan.

---

## 🎯 Maqsad

Jamoaviy ishni samarali tashkil qilish va task monitoringini avtomatlashtirish.

---

## 🚀 Asosiy funksiyalar

- 🔧 Task yaratish, yangilash va o‘chirish
- ⏰ Deadline belgilash
- 👥 Taskga user biriktirish
- 💬 Comment yozish
- 📌 Task holatlari: `TODO`, `IN_PROGRESS`, `DONE`
- 🔒 Autentifikatsiya va avtorizatsiya ()
- 🧱 REST API interfeysi
- 🛡️ Filterlar orqali xavfsizlik va log yuritish
